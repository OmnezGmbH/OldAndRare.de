# OldAndRare.de

Whisky Sammlung Ankauf Platform - Deutschland

## Über das Projekt
OldAndRare.de ist eine Plattform für den Ankauf von rare und alte Whisky-Sammlungen aus Privatbesitz.

## Phasen

### Phase 1: Landing Page (Aktuell)
- SEO-optimierte Landing Page (Deutsch)
- Kontaktformular für Bewertungsanfragen
- Trust-Building Elemente

### Phase 2: Marketplace (Geplant)
- Whisky Exchange Platform
- User Accounts
- Bewertungssystem

### Phase 3: Multi-Language (Geplant)
- Deutsch ✓
- English
- Türkçe

## Technologien
- HTML5
- CSS3
- Git & GitHub
- Vercel Hosting
